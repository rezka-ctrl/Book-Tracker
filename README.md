# Book Tracker — Трекер прочитанных книг

Графическое приложение для управления списком прочитанных книг с фильтрацией,
сохранением данных в JSON и тестами.

**Автор: Корнельзина Ульяна**

---

## Возможности

- Форма добавления книги: название, автор, жанр, количество страниц
- Кнопка «Добавить книгу» с добавлением в таблицу
- Фильтрация по жанру и по количеству страниц (больше значения)
- Сохранение данных в `data.json` и загрузка при следующем запуске
- Валидация: все поля обязательны, количество страниц — положительное целое

---

## Структура проекта

```
book-tracker/
├── main.py          # GUI-приложение (tkinter)
├── validator.py     # Валидация страниц и непустых полей
├── storage.py       # Сохранение / загрузка данных (JSON)
├── tests.py         # Юнит-тесты (unittest)
├── requirements.txt
└── .gitignore
```

---

## Установка и запуск

### Windows

```bash
python main.py
```

### Linux

```bash
sudo apt install python3-tk
python3 main.py
```

---

## Запуск тестов

```bash
python -m unittest tests -v
```

15 тестов покрывают:
- валидацию количества страниц
- валидацию непустых полей
- сохранение / загрузку JSON
- логику фильтрации по жанру и страницам

---

## Примеры использования

1. Запустить `python main.py`
2. Ввести название `Дюна`, автора `Герберт`, жанр `Фантастика`, страниц `700`
3. Нажать **Добавить книгу** — запись появится в таблице
4. Для фильтра выбрать жанр или указать минимальное количество страниц
5. Данные сохраняются автоматически
